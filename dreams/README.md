# 梦境记录

> 奇奇怪怪的梦，醒来能记多少算多少。不追求解读，先留下来。

---

## 关于这个文件夹

专门存放梦境口述与整理稿。梦里的场景常常混搭、跳跃、细节模糊——这很正常。记录的意义在于：

- **留存** — 很多梦醒来半小时就忘了
- **对照** — 重复出现的场景、人物、情绪，日后可以回看
- **好奇** — 身体在睡、意识在演，本身就够有意思

---

## 命名约定

| 格式 | 示例 |
|------|------|
| `序号-日期-简短标题.md` | `001-2026-07-18-大学宿舍与高中校园.md` |

序号三位数，方便按顺序翻阅；标题概括梦里最鲜明的场景即可。

---

## 每篇建议包含

1. **元信息** — 日期、入睡时段、醒来感受（可选）
2. **梦境正文** — 按场景顺序写，不必追求完整
3. **醒后想法** — 对梦本身的观察、联想、待验证的事
4. **遗忘备注** — 记不清的地方如实标注

不必每篇都写满，有几段就记几段。
