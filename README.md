# life

> 工作之外的时间，心里的想法，日常生活的点点滴滴 —— 都在这里被记录下来。

这不是一个项目仓库，而是一本**持续书写的生活日志**。像一台不会关机的摄像头，安静地捕捉那些容易被遗忘的时刻：一次散步、一顿晚饭、一段独处、一个突然冒出来的念头。

---

## 关于这个仓库

生活不只有代码和任务清单。这个仓库存在的意义，是给自己留一个可以回望的地方。

- **日常** — 工作以外做过的事、去过的地方、遇见的人
- **想法** — 情绪、感悟、随想，不必完整，真实就好
- **时刻** — 值得记住的片段，哪怕只是一句话

没有固定的更新节奏，也没有必须遵守的格式。写下来了，就算数。

---

## 记录方式

可以用任何你觉得舒服的方式：

| 方式 | 说明 |
|------|------|
| 按日期归档 | 例如 `2026/07/05.md`，一天一篇或几篇 |
| 按主题整理 | 例如 `travel/`、`reading/`、`thoughts/` |
| 随手一记 | 根目录下一个文件、几行字，也完全没问题 |

Markdown 足够简单，也足够长久。文字、图片、链接，想放什么就放什么。

---

## 为什么放在 GitHub

- **持久** — 不依赖某个 App 或平台是否还在
- **公开成长** — 记录成败，不记录隐私；不怕丢人，但要记得教训
- **有迹可循** — 提交历史本身，也是生活走过的痕迹

## 一点约定（给自己）

1. 不必追求完美，先写下来
2. 可以很短，一两句话也有价值
3. 想写就写，不想写就休息
4. 回头看时，不评判当时的自己
